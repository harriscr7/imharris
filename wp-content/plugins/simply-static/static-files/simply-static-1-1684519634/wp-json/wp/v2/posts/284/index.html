Ifstream Class:<\/h2>\n\n\n\n

ifstream<\/code> is a class in C++ that stands for “input file stream.” It is used for reading data from files.<\/p>\n\n\n\n

To use ifstream<\/code>, you need to include the <fstream><\/code> header file in your C++ program. Here’s a simple example that demonstrates how to use ifstream<\/code>:<\/p>\n\n\n\n

#include <fstream>\n#include <iostream>\nusing namespace std;\nint main() {\n    ifstream inputFile(\"example.txt\");  \/\/ Open the file \"example.txt\" for reading\n\n    if (inputFile.is_open()) {  \/\/ Check if the file was successfully opened\n        string line;\n        while (getline(inputFile, line)) {  \/\/ Read each line from the file\n            cout << line << endl;  \/\/ Output the line to the console\n        }\n\n        inputFile.close();  \/\/ Close the file\n    } else {\n        cout << \"Unable to open the file.\" << endl;\n    }\n\n    return 0;\n}\n<\/code><\/pre>\n\n\n\n
In this example, we create an ifstream<\/code> object named inputFile<\/code> and open the file “example.txt” for reading. We then check if the file was successfully opened using the is_open()<\/code> function. If the file is open, we use a while<\/code> loop to read each line from the file using std::getline()<\/code>, and then output each line to the console using std::cout<\/code>. Finally, we close the file using the close()<\/code> function.<\/p>\n\n\n\n
If the file cannot be opened, an error message is displayed. Remember to provide the correct file path and ensure that the file exists in the specified location.<\/p>\n\n\n\n
Ofstream Class:<\/h2>\n\n\n\n
ofstream<\/code> is a class in C++ that stands for “output file stream.” It is used for writing data to files.<\/p>\n\n\n\n
To use ofstream<\/code>, you need to include the <fstream><\/code> header file in your C++ program. Here’s a simple example that demonstrates how to use ofstream<\/code>:<\/p>\n\n\n\n
#include <fstream>\n#include <iostream>\nusing namespace std;\nint main() {\n    ofstream outputFile(\"example.txt\");  \/\/ Open\/create the file \"example.txt\" for writing\n\n    if (outputFile.is_open()) {  \/\/ Check if the file was successfully opened\/created\n        outputFile << \"Hello, world!\" << endl;  \/\/ Write a string to the file\n        outputFile.close();  \/\/ Close the file\n    } else {\n        cout << \"Unable to open\/create the file.\" << endl;\n    }\n\n    return 0;\n}\n<\/code><\/pre>\n\n\n\n
In this example, we create an ofstream<\/code> object named outputFile<\/code> and open\/create the file “example.txt” for writing. We then check if the file was successfully opened\/created using the is_open()<\/code> function. If the file is open, we can write data to it using the <<<\/code> operator, just like we would with std::cout<\/code>. In this case, we write the string “Hello, world!” to the file. Finally, we close the file using the close()<\/code> function.<\/p>\n\n\n\n
