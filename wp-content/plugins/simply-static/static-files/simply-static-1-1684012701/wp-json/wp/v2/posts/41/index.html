C++ Output<\/h2>\n\n\n\n

In C++, cout<\/code> sends formatted output to standard output devices, such as the screen. We use the cout<\/code> object along with the <<<\/code> operator for displaying output.<\/p>\n\n\n\n

\n\n\n\n

How does this program work?<\/strong><\/p>\n\n\n\n
    \n
  • We first include the iostream<\/code> header file that allows us to display output.<\/li>\n\n\n\n
  • The cout<\/code> object is defined inside the std<\/code> namespace. To use the std<\/code> namespace, we used the using namespace std;<\/code> statement.<\/li>\n\n\n\n
  • Every C++ program starts with the main()<\/code> function. The code execution begins from the start of the main()<\/code> function.<\/li>\n\n\n\n
  • cout<\/code> is an object that prints the string inside quotation marks \" \"<\/code>. It is followed by the <<<\/code> operator.<\/li>\n\n\n\n
  • return 0;<\/code> is the “exit status” of the main()<\/code> function. The program ends with this statement, however, this statement is not mandatory.<\/li>\n<\/ul>\n\n\n\n
    Note:<\/strong> If we don’t include the using namespace std;<\/code> statement, we need to use std::cout<\/code> instead of cout<\/code>.<\/p>\n\n\n\n
