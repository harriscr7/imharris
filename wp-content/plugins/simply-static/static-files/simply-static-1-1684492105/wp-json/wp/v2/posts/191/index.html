In this tutorial, we will learn about the C++ for loop and its working with the help of some examples.<\/p>\n\n\n\n

In computer programming, loops are used to repeat a block of code.<\/p>\n\n\n\n

For example, let’s say we want to show a message 100 times. Then instead of writing the print statement 100 times, we can use a loop.<\/p>\n\n\n\n

That was just a simple example; we can achieve much more efficiency and sophistication in our programs by making effective use of loops.<\/p>\n\n\n\n

There are 3 types of loops in C++.<\/p>\n\n\n\n

    \n
  • for<\/code> loop<\/li>\n\n\n\n
  • while<\/code> loop<\/li>\n\n\n\n
  • do...while<\/code> loop<\/li>\n<\/ul>\n\n\n\n

    This tutorial focuses on C++ for<\/code> loop. We will learn about the other type of loops in the upcoming tutorials.<\/p>\n\n\n\n

    C++ for loop<\/h2>\n\n\n\n

    The syntax of for-loop is:<\/p>\n\n\n\n

    for (initialization; condition; update) {\n    \/\/ body of-loop \n}<\/code><\/pre>\n\n\n\n
    Here,<\/p>\n\n\n\n
      \n
    • initialization<\/code> – initializes variables and is executed only once<\/li>\n\n\n\n
    • condition<\/code> – if true<\/code>, the body of for<\/code> loop is executed
      if false<\/code>, the for loop is terminated<\/li>\n\n\n\n
    • update<\/code> – updates the value of initialized variables and again checks the condition<\/li>\n<\/ul>\n\n\n\n
