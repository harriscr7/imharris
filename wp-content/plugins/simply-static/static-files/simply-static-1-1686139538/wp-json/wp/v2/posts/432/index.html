In this tutorial, you’ll learn to handle strings in C++. You’ll learn to declare them, initialize them and use them for various input\/output operations.<\/p>\n\n\n\n

String is a collection of characters. There are two types of strings commonly used in C++ programming language:<\/p>\n\n\n\n

    \n
  • Strings that are objects of string class (The Standard C++ Library string class)<\/li>\n\n\n\n
  • C-strings (C-style Strings)<\/li>\n<\/ul>\n\n\n\n
    \n\n\n\n

    C-strings<\/h2>\n\n\n\n

    In C programming, the collection of characters is stored in the form of arrays. This is also supported in C++ programming. Hence it’s called C-strings.<\/p>\n\n\n\n

    C-strings are arrays of type char<\/code> terminated with null character, that is, \\0<\/code> (ASCII value of null character is 0).<\/p>\n\n\n\n

    \n\n\n\n

    How to define a C-string?<\/h3>\n\n\n\n
    char str[] = \"C++\";<\/pre>\n\n\n\n
    In the above code, str<\/code> is a string and it holds 4 characters.<\/p>\n\n\n\n
    Although, “C++<\/code>” has 3 character, the null character \\0<\/code> is added to the end of the string automatically.<\/p>\n\n\n\n
    \n\n\n\n
    Alternative ways of defining a string<\/h3>\n\n\n\n
    char str[4] = \"C++\";\n     \nchar str[] = {'C','+','+','\\0'};\n\nchar str[4] = {'C','+','+','\\0'};<\/pre>\n\n\n\n
    Like arrays, it is not necessary to use all the space allocated for the string. For example:<\/p>\n\n\n\n
    char str[100] = \"C++\";<\/pre>\n\n\n\n
    \n\n\n\n
    Example 1: C++ String to read a word<\/h3>\n\n\n\n
    C++ program to display a string entered by user.<\/strong><\/p>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nint main()\n{\n    char str[100];\n\n    cout << \"Enter a string: \";\n    cin >> str;\n    cout << \"You entered: \" << str << endl;\n\n    cout << \"\\nEnter another string: \";\n    cin >> str;\n    cout << \"You entered: \"<<str<<endl;\n\n    return 0;\n}<\/code><\/pre>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    Enter a string: C++\nYou entered: C++\n\nEnter another string: Programming is fun.\nYou entered: Programming<\/samp>\n<\/pre>\n\n\n\n
    Notice that, in the second example only “Programming” is displayed instead of “Programming is fun”.<\/p>\n\n\n\n
    This is because the extraction operator >> works as scanf()<\/code> in C and considers a space ” ” has a terminating character.<\/p>\n\n\n\n
    \n\n\n\n
    Example 2: C++ String to read a line of text<\/h3>\n\n\n\n
    C++ program to read and display an entire line entered by user.<\/strong><\/p>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nint main()\n{\n    char str[100];\n    cout << \"Enter a string: \";\n    cin.get(str, 100);\n\n    cout << \"You entered: \" << str << endl;\n    return 0;\n}<\/code><\/pre>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    Enter a string: Programming is fun.\nYou entered: Programming is fun.\n<\/samp><\/pre>\n\n\n\n
    To read the text containing blank space, cin.get<\/code> function can be used. This function takes two arguments.<\/p>\n\n\n\n
    First argument is the name of the string (address of first element of string) and second argument is the maximum size of the array.<\/p>\n\n\n\n
    In the above program, str<\/var> is the name of the string and 100<\/code> is the maximum size of the array.<\/p>\n\n\n\n
    \n\n\n\n
    string Object<\/h2>\n\n\n\n
    In C++, you can also create a string object for holding strings.<\/p>\n\n\n\n
    Unlike using char arrays, string objects has no fixed length, and can be extended as per your requirement.<\/p>\n\n\n\n
    \n\n\n\n
    Example 3: C++ string using string data type<\/h3>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nint main()\n{\n    \/\/ Declaring a string object\n    string str;\n    cout << \"Enter a string: \";\n    getline(cin, str);\n\n    cout << \"You entered: \" << str << endl;\n    return 0;\n}<\/code><\/pre>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    Enter a string: Programming is fun.\nYou entered: Programming is fun.\n<\/samp><\/pre>\n\n\n\n
    In this program, a string str<\/var> is declared. Then the string is asked from the user.<\/p>\n\n\n\n
    Instead of using cin>><\/code> or cin.get()<\/code> function, you can get the entered line of text using getline()<\/code>.<\/p>\n\n\n\n
    getline()<\/code> function takes the input stream as the first parameter which is cin<\/code> and str<\/code> as the location of the line to be stored.<\/p>\n\n\n\n
    \n\n\n\n
    Passing String to a Function<\/h2>\n\n\n\n
    Strings are passed to a function in a similar way\u00a0arrays are passed to a function.<\/p>\n\n\n\n
    #include <iostream>\n\nusing namespace std;\n\nvoid display(char *);\nvoid display(string);\n\nint main()\n{\n    string str1;\n    char str[100];\n\n    cout << \"Enter a string: \";\n    getline(cin, str1);\n\n    cout << \"Enter another string: \";\n    cin.get(str, 100, '\\n');\n\n    display(str1);\n    display(str);\n    return 0;\n}\n\nvoid display(char s[])\n{\n    cout << \"Entered char array is: \" << s << endl;\n}\n\nvoid display(string s)\n{\n    cout << \"Entered string is: \" << s << endl;\n}<\/code><\/pre>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    Enter a string:  Programming is fun.\nEnter another string:  Really?\nEntered string is: Programming is fun.\nEntered char array is: Really?<\/samp><\/pre>\n\n\n\n
    In the above program, two strings are asked to enter. These are stored in str<\/var> and str1<\/var> respectively, where str is a char<\/code> array and str1 is a string<\/code> object.<\/p>\n\n\n\n
    Then, we have two functions display()<\/code> that outputs the string onto the string.<\/p>\n\n\n\n
    The only difference between the two functions is the parameter. The first display()<\/code> function takes char array as a parameter, while the second takes string as a parameter.<\/p>\n\n\n\n
