Passing structure to function in C++<\/h2>\n\n\n\n

A structure variable can be passed to a function in similar way as normal argument. Consider this example:<\/p>\n\n\n\n

\n\n\n\n

Example 1: C++ Structure and Function<\/h3>\n\n\n\n
#include <iostream>\nusing namespace std;\n\nstruct Person {\n    char name[50];\n    int age;\n    float salary;\n};\n\nvoid displayData(Person);   \/\/ Function declaration\n\nint main() {\n    Person p;\n\n    cout << \"Enter Full name: \";\n    cin.get(p.name, 50);\n    cout << \"Enter age: \";\n    cin >> p.age;\n    cout << \"Enter salary: \";\n    cin >> p.salary;\n\n    \/\/ Function call with structure variable as an argument\n    displayData(p);\n\n    return 0;\n}\n\nvoid displayData(Person p) {\n    cout << \"\\nDisplaying Information.\" << endl;\n    cout << \"Name: \" << p.name << endl;\n    cout <<\"Age: \" << p.age << endl;\n    cout << \"Salary: \" << p.salary;\n}<\/code><\/pre>\n\n\n\n
Output<\/strong><\/p>\n\n\n\n
Enter Full name: Bill Jobs\nEnter age: 55\nEnter salary: 34233.4\n\nDisplaying Information.\nName: Bill Jobs\nAge: 55\nSalary: 34233.4<\/samp><\/pre>\n\n\n\n
In this program, user is asked to enter the name<\/var>, age<\/var> and salary<\/var> of a Person inside main()<\/code> function.<\/p>\n\n\n\n
Then, the structure variable p<\/var> is to passed to a function using.<\/p>\n\n\n\n
displayData(p);<\/pre>\n\n\n\n
The return type of displayData()<\/code> is void<\/code> and a single argument of type structure Person<\/var> is passed.<\/p>\n\n\n\n
Then the members of structure p<\/code> is displayed from this function.<\/p>\n\n\n\n

Example 2: Returning structure from function in C++<\/h3>\n\n\n\n
#include <iostream>\nusing namespace std;\n\nstruct Person {\n    char name[50];\n    int age;\n    float salary;\n};\n\nPerson getData(Person); \nvoid displayData(Person); \n\nint main() {\n\n    Person p, temp;\n\n    temp = getData(p);\n    p = temp;\n    displayData(p);\n\n    return 0;\n}\n\nPerson getData(Person p) {\n\n    cout << \"Enter Full name: \";\n    cin.get(p.name, 50);\n\n    cout << \"Enter age: \";\n    cin >> p.age;\n\n    cout << \"Enter salary: \";\n    cin >> p.salary;\n\n    return p;\n}\n\nvoid displayData(Person p) {\n    cout << \"\\nDisplaying Information.\" << endl;\n    cout << \"Name: \" << p.name << endl;\n    cout <<\"Age: \" << p.age << endl;\n    cout << \"Salary: \" << p.salary;\n}\n<\/code><\/pre>\n\n\n\n
The output of this program is the same as the program above.<\/p>\n\n\n\n
In this program, we have created two structure variables p<\/var> and temp<\/var> of type Person<\/var><\/code> under the main()<\/code> function.<\/p>\n\n\n\n
The structure variable p<\/var> is passed to getData()<\/code> function which takes input from the user which is then stored in the temp<\/var> variable.<\/p>\n\n\n\n
temp = getData(p); <\/pre>\n\n\n\n
We then assign the value of temp<\/var> to p<\/var>.<\/p>\n\n\n\n
p = temp;<\/pre>\n\n\n\n
Then the structure variable p<\/var> is passed to displayData()<\/code> function, which displays the information.<\/p>\n\n\n\n
Note:<\/strong> We don’t really need to use the temp<\/var> variable for most compilers and C++ versions. Instead, we can simply use the following code:<\/p>\n\n\n\n
