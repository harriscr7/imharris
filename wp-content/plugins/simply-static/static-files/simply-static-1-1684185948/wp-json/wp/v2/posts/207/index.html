In this tutorial, you will learn about different approaches you can take to solve a single problem using functions.<\/p>\n\n\n\n

For better understanding of arguments and return in functions, user-defined functions can be categorised as:<\/p>\n\n\n\n

    \n
  • Function with no argument and no return value<\/a><\/li>\n\n\n\n
  • Function with no argument but return value<\/a><\/li>\n\n\n\n
  • Function with argument but no return value<\/a><\/li>\n\n\n\n
  • Function with argument and return value<\/a><\/li>\n<\/ul>\n\n\n\n

    Consider a situation in which you have to check prime number. This problem is solved below by making user-defined function in 4 different ways as mentioned above.<\/p>\n\n\n\n

    \n\n\n\n

    <\/a>Example 1: No arguments passed and no return value<\/h2>\n\n\n\n
    # include <iostream>\nusing namespace std;\n\nvoid prime();\n\nint main()\n{\n    \/\/ No argument is passed to prime()\n    prime();\n    return 0;\n}\n\n\n\/\/ Return type of function is void because value is not returned.\nvoid prime()\n{\n\n    int num, i, flag = 0;\n\n    cout << \"Enter a positive integer enter to check: \";\n    cin >> num;\n\n    for(i = 2; i <= num\/2; ++i)\n    {\n        if(num % i == 0)\n        {\n            flag = 1; \n            break;\n        }\n    }\n\n    if (flag == 1)\n    {\n        cout << num << \" is not a prime number.\";\n    }\n    else\n    {\n        cout << num << \" is a prime number.\";\n    }\n}\n<\/code><\/pre>\n\n\n\n
    In the above program, prime()<\/code> is called from the main() with no arguments.<\/p>\n\n\n\n
    prime()<\/code> takes the positive number from the user and checks whether the number is a prime number or not.<\/p>\n\n\n\n
    Since, return type of prime()<\/code> is void<\/code>, no value is returned from the function.<\/p>\n\n\n\n
    \n\n\n\n
    <\/a>Example 2: No arguments passed but a return value<\/h2>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nint prime();\n\nint main()\n{\n    int num, i, flag = 0;\n\n    \/\/ No argument is passed to prime()\n    num = prime();\n    for (i = 2; i <= num\/2; ++i)\n    {\n        if (num%i == 0)\n        {\n            flag = 1;\n            break;\n        }\n    }\n\n    if (flag == 1)\n    {\n        cout<<num<<\" is not a prime number.\";\n    }\n    else\n    {\n        cout<<num<<\" is a prime number.\";\n    }\n    return 0;\n}\n\n\/\/ Return type of function is int\nint prime()\n{\n    int n;\n\n    printf(\"Enter a positive integer to check: \");\n    cin >> n;\n\n    return n;\n}<\/code><\/pre>\n\n\n\n
    In the above program, prime()<\/code> function is called from the main()<\/code> with no arguments.<\/p>\n\n\n\n
    prime()<\/code> takes a positive integer from the user. Since, return type of the function is an int<\/code>, it returns the inputted number from the user back to the calling main()<\/code> function.<\/p>\n\n\n\n
    Then, whether the number is prime or not is checked in the main() itself and printed onto the screen.<\/p>\n\n\n\n
    \n\n\n\n
    <\/a>Example 3: Arguments passed but no return value<\/h2>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nvoid prime(int n);\n\nint main()\n{\n    int num;\n    cout << \"Enter a positive integer to check: \";\n    cin >> num;\n    \n    \/\/ Argument num is passed to the function prime()\n    prime(num);\n    return 0;\n}\n\n\/\/ There is no return value to calling function. Hence, return type of function is void. *\/\nvoid prime(int n)\n{\n    int i, flag = 0;\n    for (i = 2; i <= n\/2; ++i)\n    {\n        if (n%i == 0)\n        {\n            flag = 1;\n            break;\n        }\n    }\n\n    if (flag == 1)\n    {\n        cout << n << \" is not a prime number.\";\n    }\n    else {\n        cout << n << \" is a prime number.\";\n    }\n}<\/code><\/pre>\n\n\n\n
    In the above program, positive number is first asked from the user which is stored in the variable num<\/var>.<\/p>\n\n\n\n
    Then, num<\/var> is passed to the prime()<\/code> function where, whether the number is prime or not is checked and printed.<\/p>\n\n\n\n
    Since, the return type of prime()<\/code> is a void<\/code>, no value is returned from the function.<\/p>\n\n\n\n
    \n\n\n\n
    <\/a>Example 4: Arguments passed and a return value.<\/h2>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\nint prime(int n);\n\nint main()\n{\n    int num, flag = 0;\n    cout << \"Enter positive integer to check: \";\n    cin >> num;\n\n    \/\/ Argument num is passed to check() function\n    flag = prime(num);\n\n    if(flag == 1)\n        cout << num << \" is not a prime number.\";\n    else\n        cout<< num << \" is a prime number.\";\n    return 0;\n}\n\n\/* This function returns integer value.  *\/\nint prime(int n)\n{\n    int i;\n    for(i = 2; i <= n\/2; ++i)\n    {\n        if(n % i == 0)\n            return 1;\n    }\n\n    return 0;\n}\n<\/code><\/pre>\n\n\n\n
    In the above program, a positive integer is asked from the user and stored in the variable num<\/code>.<\/p>\n\n\n\n
    Then, num<\/code> is passed to the function prime()<\/code> where, whether the number is prime or not is checked.<\/p>\n\n\n\n
    Since, the return type of prime()<\/code> is an int<\/code>, 1 or 0 is returned to the main()<\/code> calling function. If the number is a prime number, 1 is returned. If not, 0 is returned.<\/p>\n\n\n\n
    Back in the main()<\/code> function, the returned 1 or 0 is stored in the variable flag<\/var>, and the corresponding text is printed onto the screen.<\/p>\n\n\n\n
    \n\n\n\n
    Which method is better?<\/h2>\n\n\n\n
    All four programs above gives the same output and all are technically correct program.<\/p>\n\n\n\n
    There is no hard and fast rule on which method should be chosen.<\/p>\n\n\n\n
    The particular method is chosen depending upon the situation and how you want to solve a problem.<\/p>\n","protected":false},"excerpt":{"rendered":"
