In this article, you’ll learn how to return a value by reference in a function and use it efficiently in your program.<\/p>\n\n\n\n

In C++ Programming, not only can you pass values by reference to a function<\/a> but you can also return a value by reference.<\/p>\n\n\n\n

To understand this feature, you should have the knowledge of:<\/p>\n\n\n\n

    \n
  • Global variables<\/a><\/li>\n<\/ul>\n\n\n\n
    \n\n\n\n

    Example: Return by Reference<\/h2>\n\n\n\n
    #include <iostream>\nusing namespace std;\n\n\/\/ global variable\nint num;\n\n\/\/ function declaration\nint& test();\n\nint main() {\n\n  \/\/ assign 5 to num variable\n  \/\/ equivalent to num = 5;\n  test() = 5;\n\n  cout << num;\n\n  return 0;\n}\n\n\/\/ function definition\n\/\/ returns the address of num variable\nint& test() {\n  return num;\n}<\/code><\/pre>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    5<\/samp><\/pre>\n\n\n\n
    In program above, the return type of function test()<\/code> is int&<\/code>. Hence, this function returns a reference of the variable num<\/var>.<\/p>\n\n\n\n
    The return statement is return num;<\/code>. Unlike return by value, this statement doesn’t return value of num<\/var>, instead it returns the variable itself (address).<\/p>\n\n\n\n
    So, when the variable<\/strong> is returned, it can be assigned a value as done in test() = 5;<\/code><\/p>\n\n\n\n
    This stores 5<\/strong> to the variable num<\/var>, which is displayed onto the screen.<\/p>\n\n\n\n
    \n\n\n\n
    Important Things to Remember When Returning by Reference.<\/h3>\n\n\n\n
      \n
    • Ordinary function returns value but this function doesn’t. Hence, you cannot return a constant from the function.int& test() { return 2; }<\/li>\n\n\n\n
    • You cannot return a local variable from this function.int& test() { int n = 2; return n; }<\/li>\n<\/ul>\n","protected":false},"excerpt":{"rendered":"
