In this article, you’ll learn about goto statment, how it works and why should it be avoided.<\/p>\n\n\n\n

In C++ programming, the goto statement is used for altering the normal sequence of program execution by transferring control to some other part of the program.<\/p>\n\n\n\n

Syntax of goto Statement<\/h2>\n\n\n\n
goto label;\n... .. ...\n... .. ...\n... .. ...\nlabel: \nstatement;\n... .. ...<\/pre>\n\n\n\n
In the syntax above, label<\/var> is an identifier. When goto label;<\/code> is encountered, the control of program jumps to label:<\/code> and executes the code below it.<\/p>\n\n\n\n
