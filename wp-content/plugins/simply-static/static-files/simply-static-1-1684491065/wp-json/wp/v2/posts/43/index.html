In this tutorial, we will learn about the basics of C++ type conversion with the help of examples.<\/p>\n\n\n\n

C++ allows us to convert data of one type to that of another. This is known as type conversion.<\/p>\n\n\n\n

There are two types of type conversion in C++.<\/p>\n\n\n\n

    \n
  1. Implicit Conversion<\/li>\n\n\n\n
  2. Explicit Conversion (also known as Type Casting)<\/li>\n<\/ol>\n\n\n\n
    \n\n\n\n

    Implicit Type Conversion<\/h2>\n\n\n\n

    The type conversion that is done automatically done by the compiler is known as implicit type conversion. This type of conversion is also known as automatic conversion.<\/p>\n\n\n\n

    Let us look at two examples of implicit type conversion.<\/p>\n\n\n\n

    \n\n\n\n

    Example 1: Conversion From int to double<\/h3>\n\n\n\n
    \/\/ Working of implicit type-conversion\n\n#include <iostream>\nusing namespace std;\n\nint main() {\n   \/\/ assigning an int value to num_int\n   int num_int = 9;\n\n   \/\/ declaring a double type variable\n   double num_double;\n \n   \/\/ implicit conversion\n   \/\/ assigning int value to a double variable\n   num_double = num_int;\n\n   cout << \"num_int = \" << num_int << endl;\n   cout << \"num_double = \" << num_double << endl;\n\n   return 0;\n}<\/code><\/pre>\n\n\n\n
    Run Code<\/a><\/p>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    num_int = 9\nnum_double = 9<\/samp><\/pre>\n\n\n\n
    In the program, we have assigned an int<\/code> data to a double<\/code> variable.<\/p>\n\n\n\n
    num_double = num_int;<\/code><\/pre>\n\n\n\n
    Here, the int<\/code> value is automatically converted to double<\/code> by the compiler before it is assigned to the num_double<\/var> variable. This is an example of implicit type conversion.<\/p>\n\n\n\n
    \n\n\n\n
    Example 2: Automatic Conversion from double to int<\/h3>\n\n\n\n
    \/\/Working of Implicit type-conversion\n\n#include <iostream>\nusing namespace std;\n\nint main() {\n\n   int num_int;\n   double num_double = 9.99;\n\n   \/\/ implicit conversion\n   \/\/ assigning a double value to an int variable\n   num_int = num_double;\n\n   cout << \"num_int = \" << num_int << endl;\n   cout << \"num_double = \" << num_double << endl;\n\n   return 0;\n}<\/code><\/pre>\n\n\n\n
    Run Code<\/a><\/p>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    num_int = 9\nnum_double = 9.99<\/samp><\/pre>\n\n\n\n
    In the program, we have assigned a double<\/code> data to an int<\/code> variable.<\/p>\n\n\n\n
    num_int = num_double;<\/code><\/pre>\n\n\n\n
    Here, the double<\/code> value is automatically converted to int<\/code> by the compiler before it is assigned to the num_int<\/var> variable. This is also an example of implicit type conversion.<\/p>\n\n\n\n
    Note:<\/strong> Since int<\/code> cannot have a decimal part, the digits after the decimal point are truncated in the above example.<\/p>\n\n\n\n
    \n\n\n\n
    Data Loss During Conversion (Narrowing Conversion)<\/h3>\n\n\n\n
    As we have seen from the above example, conversion from one data type to another is prone to data loss. This happens when data of a larger type is converted to data of a smaller type.<\/p>\n\n\n\n
