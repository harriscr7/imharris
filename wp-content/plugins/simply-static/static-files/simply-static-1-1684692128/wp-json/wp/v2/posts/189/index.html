In this tutorial, we will learn about switch statement and its working in C++ programming with the help of some examples.<\/p>\n\n\n\n

The switch<\/code> statement allows us to execute a block of code among many alternatives.<\/p>\n\n\n\n

The syntax of the switch<\/code> statement in C++ is:<\/p>\n\n\n\n

switch (expression)  {\n    case constant1:\n        \/\/ code to be executed if \n        \/\/ expression is equal to constant1;\n        break;\n\n    case constant2:\n        \/\/ code to be executed if\n        \/\/ expression is equal to constant2;\n        break;\n        .\n        .\n        .\n    default:\n        \/\/ code to be executed if\n        \/\/ expression doesn't match any constant\n}<\/code><\/pre>\n\n\n\n
How does the switch statement work?<\/strong><\/p>\n\n\n\n
The expression<\/code> is evaluated once and compared with the values of each case<\/code> label.<\/p>\n\n\n\n
    \n
  • If there is a match, the corresponding code after the matching label is executed. For example, if the value of the variable is equal to\u00a0constant2<\/code>, the code after\u00a0case constant2:<\/code>\u00a0is executed until the\u00a0break statement\u00a0is encountered.<\/li>\n\n\n\n
  • If there is no match, the code after default:<\/code> is executed.<\/li>\n<\/ul>\n\n\n\n
    Note<\/strong>: We can do the same thing with the if...else..if<\/code> ladder. However, the syntax of the switch<\/code> statement is cleaner and much easier to read and write.<\/p>\n\n\n\n
