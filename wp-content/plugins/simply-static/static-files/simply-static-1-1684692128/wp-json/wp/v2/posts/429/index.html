In this tutorial, we will learn how to pass a single-dimensional and multidimensional array as a function parameter in C++ with the help of examples.<\/p>\n\n\n\n

In C++, we can pass arrays as an argument to a function. And, also we can return arrays from a function.<\/p>\n\n\n\n

Before you learn about passing arrays as a function argument, make sure you know about\u00a0C++ Arrays\u00a0and\u00a0C++ Functions.<\/p>\n\n\n\n

\n\n\n\n

Syntax for Passing Arrays as Function Parameters<\/h2>\n\n\n\n

The syntax for passing an array to a function is:<\/p>\n\n\n\n

returnType functionName(dataType arrayName[arraySize]) {\n    \/\/ code\n}<\/code><\/pre>\n\n\n\n
Let’s see an example,<\/p>\n\n\n\n
int total(int marks[5]) {\n    \/\/ code\n}<\/code><\/pre>\n\n\n\n
Here, we have passed an int<\/code> type array named marks<\/var> to the function total()<\/code>. The size of the array is 5<\/var>.<\/p>\n\n\n\n
\n\n\n\n
Example 1: Passing One-dimensional Array to a Function<\/h3>\n\n\n\n
\/\/ C++ Program to display marks of 5 students\n\n#include <iostream>\nusing namespace std;\n\n\/\/ declare function to display marks\n\/\/ take a 1d array as parameter\nvoid display(int m[5]) {\n    cout << \"Displaying marks: \" << endl;\n\n    \/\/ display array elements    \n    for (int i = 0; i < 5; ++i) {\n        cout << \"Student \" << i + 1 << \": \" << m[i] << endl;\n    }\n}\n\nint main() {\n\n    \/\/ declare and initialize an array\n    int marks[5] = {88, 76, 90, 61, 69};\n    \n    \/\/ call display function\n    \/\/ pass array as argument\n    display(marks);\n\n    return 0;\n}<\/code><\/pre>\n\n\n\n
Run Code<\/a><\/p>\n\n\n\n
Output<\/strong><\/p>\n\n\n\n
Displaying marks: \nStudent 1: 88\nStudent 2: 76\nStudent 3: 90\nStudent 4: 61\nStudent 5: 69<\/samp><\/pre>\n\n\n\n
Here,<\/p>\n\n\n\n
    \n
  1. When we call a function by passing an array as the argument, only the name of the array is used.display(marks);<\/code>Here, the argument marks<\/var> represent the memory address of the first element of array marks[5]<\/var>.<\/li>\n\n\n\n
  2. However, notice the parameter of the display()<\/code> function.void display(int m[5])<\/code>Here, we use the full declaration of the array in the function parameter, including the square braces []<\/code>.<\/li>\n\n\n\n
  3. The function parameter int m[5]<\/code> converts to int* m;<\/code>. This points to the same address pointed by the array marks<\/var>. This means that when we manipulate m[5]<\/var> in the function body, we are actually manipulating the original array marks<\/var>.

    C++ handles passing an array to a function in this way to save memory and time.<\/li>\n<\/ol>\n\n\n\n

    Passing Multidimensional Array to a Function<\/h3>\n\n\n\n
    We can also pass\u00a0Multidimensional arrays\u00a0as an argument to the function. For example,<\/p>\n\n\n\n
    Example 2: Passing Multidimensional Array to a Function<\/h3>\n\n\n\n
    \/\/ C++ Program to display the elements of two\n\/\/ dimensional array by passing it to a function\n\n#include <iostream>\nusing namespace std;\n\n\/\/ define a function \n\/\/ pass a 2d array as a parameter\nvoid display(int n[][2]) {\n    cout << \"Displaying Values: \" << endl;\n    for (int i = 0; i < 3; ++i) {\n        for (int j = 0; j < 2; ++j) {\n            cout << \"num[\" << i << \"][\" << j << \"]: \" << n[i][j] << endl;\n        }\n    }\n}\n\nint main() {\n\n    \/\/ initialize 2d array\n    int num[3][2] = {\n        {3, 4},\n        {9, 5},\n        {7, 1}\n    };\n\n    \/\/ call the function\n    \/\/ pass a 2d array as an argument\n    display(num);\n\n    return 0;\n}\n<\/strong><\/code><\/pre>\n\n\n\n
    Run Code<\/a><\/p>\n\n\n\n
    Output<\/strong><\/p>\n\n\n\n
    Displaying Values: \nnum[0][0]: 3\nnum[0][1]: 4\nnum[1][0]: 9\nnum[1][1]: 5\nnum[2][0]: 7\nnum[2][1]: 1<\/samp><\/pre>\n\n\n\n
    In the above program, we have defined a function named display()<\/code>. The function takes a two dimensional array, int n[][2]<\/code> as its argument and prints the elements of the array.<\/p>\n\n\n\n
    While calling the function, we only pass the name of the two dimensional array as the function argument display(num)<\/code>.<\/p>\n\n\n\n
    Note<\/strong>: It is not mandatory to specify the number of rows in the array. However, the number of columns should always be specified. This is why we have used int n[][2]<\/code>.<\/p>\n\n\n\n
    We can also pass arrays with more than 2 dimensions as a function argument.<\/p>\n\n\n\n
    \n\n\n\n
    C++ Returning an Array From a Function<\/h2>\n\n\n\n
    We can also return an array from the function. However, the actual array is not returned. Instead the address of the first element of the array is returned with the help of\u00a0pointers.<\/p>\n\n\n\n
