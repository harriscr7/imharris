      C++ if Statement<\/h2>\n\n\n\n

      The syntax of the if<\/code> statement is:<\/p>\n\n\n\n

      if (condition) {\n  \/\/ body of if statement\n}<\/code><\/pre>\n\n\n\n
      The if<\/code> statement evaluates the condition<\/code> inside the parentheses ( )<\/code>.<\/p>\n\n\n\n
        \n
      • If the condition<\/code> evaluates to true<\/code>, the code inside the body of if<\/code> is executed.<\/li>\n\n\n\n
