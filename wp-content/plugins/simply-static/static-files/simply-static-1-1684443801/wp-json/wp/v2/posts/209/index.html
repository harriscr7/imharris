In this tutorial, we will learn about the function overloading in C++ with examples.<\/p>\n\n\n\n

In C++, two functions can have the same name if the number and\/or type of arguments passed is different.<\/p>\n\n\n\n

These functions having the same name but different arguments are known as overloaded functions. For example:<\/p>\n\n\n\n

\/\/ same name different arguments\nint test() { }\nint test(int a) { }\nfloat test(double a) { }\nint test(int a, double b) { }<\/code><\/pre>\n\n\n\n
Here, all 4 functions are overloaded functions.<\/p>\n\n\n\n
Notice that the return types of all these 4 functions are not the same. Overloaded functions may or may not have different return types but they must have different arguments. For example,<\/p>\n\n\n\n
\/\/ Error code\nint test(int a) { }\ndouble test(int b){ }<\/code><\/pre>\n\n\n\n
Here, both functions have the same name, the same type, and the same number of arguments. Hence, the compiler will throw an error.<\/p>\n\n\n\n

Example 1: Overloading Using Different Types of Parameter<\/h2>\n\n\n\n
\/\/ Program to compute absolute value\n\/\/ Works for both int and float\n\n#include <iostream>\nusing namespace std;\n\n\/\/ function with float type parameter\nfloat absolute(float var){\n    if (var < 0.0)\n        var = -var;\n    return var;\n}\n\n\/\/ function with int type parameter\nint absolute(int var) {\n     if (var < 0)\n         var = -var;\n    return var;\n}\n\nint main() {\n    \n    \/\/ call function with int type parameter\n    cout << \"Absolute value of -5 = \" << absolute(-5) << endl;\n\n    \/\/ call function with float type parameter\n    cout << \"Absolute value of 5.5 = \" << absolute(5.5f) << endl;\n    return 0;\n}<\/code><\/pre>\n\n\n\n
