Using fstream class:<\/strong><\/p>\n\n\n\n

To read and write data using fstream<\/code> in C++, you can follow these steps:<\/p>\n\n\n\n

    \n
  1. Include the necessary header file:<\/li>\n<\/ol>\n\n\n\n
    include <fstream>\n<\/code><\/pre>\n\n\n\n
      \n
    1. Create an instance of fstream<\/code> and open a file:<\/li>\n<\/ol>\n\n\n\n
      fstream file;\nfile.open(\"filename.txt\", ios::in | ios::out);\n<\/code><\/pre>\n\n\n\n
      You can also use different variable name, like “fstream myfile;” but than you have to replace variable name everywhere. <\/p>\n\n\n\n
      For example “myfile.open(\"filename.txt\", ios::in | ios::out);\"<\/code><\/p>\n\n\n\n
      To only read data you can only write “myfile.open(\"filename.txt\", ios::in);\"<\/code><\/p>\n\n\n\n
      To only write you can only write “myfile.open(\"filename.txt\", ios::out);\"<\/code><\/p>\n\n\n\n
      To do both what you can do is write “myfile.open(\"filename.txt\", ios::in | ios::out<\/code>);\"<\/code><\/p>\n\n\n\n
