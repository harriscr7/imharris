In this tutorial, we will learn the use of while and do…while loops in C++ programming with the help of some examples.<\/p>\n\n\n\n

In computer programming, loops are used to repeat a block of code.<\/p>\n\n\n\n

For example, let’s say we want to show a message 100 times. Then instead of writing the print statement 100 times, we can use a loop.<\/p>\n\n\n\n

That was just a simple example; we can achieve much more efficiency and sophistication in our programs by making effective use of loops.<\/p>\n\n\n\n

There are 3<\/strong> types of loops in C++.<\/p>\n\n\n\n

    \n
  1. for<\/code> loop<\/li>\n\n\n\n
  2. while<\/code> loop<\/li>\n\n\n\n
  3. do...while<\/code> loop<\/li>\n<\/ol>\n\n\n\n

    In the previous tutorial, we learned about the C++ for loop<\/a>. Here, we are going to learn about while<\/code> and do...while<\/code> loops.<\/p>\n\n\n\n

    \n\n\n\n

    C++ while Loop<\/h2>\n\n\n\n

    The syntax of the while<\/code> loop is:<\/p>\n\n\n\n

    while (condition) {\n    \/\/ body of the loop\n}<\/code><\/pre>\n\n\n\n
    Here,<\/p>\n\n\n\n
      \n
    • A while<\/code> loop evaluates the condition<\/code><\/li>\n\n\n\n
    • If the condition<\/code> evaluates to true<\/code>, the code inside the while<\/code> loop is executed.<\/li>\n\n\n\n
    • The condition<\/code> is evaluated again.<\/li>\n\n\n\n
    • This process continues until the condition<\/code> is false<\/code>.<\/li>\n\n\n\n
    • When the condition<\/code> evaluates to false<\/code>, the loop terminates.<\/li>\n<\/ul>\n\n\n\n
