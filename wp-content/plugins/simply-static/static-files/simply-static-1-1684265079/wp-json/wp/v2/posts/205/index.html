In this tutorial, we will learn about the C++ function and function expressions with the help of examples.<\/p>\n\n\n\n

A function is a block of code that performs a specific task.<\/p>\n\n\n\n

Suppose we need to create a program to create a circle and color it. We can create two functions to solve this problem:<\/p>\n\n\n\n

    \n
  • a function to draw the circle<\/li>\n\n\n\n
  • a function to color the circle<\/li>\n<\/ul>\n\n\n\n

    Dividing a complex problem into smaller chunks makes our program easy to understand and reusable.<\/p>\n\n\n\n

    There are two types of function:<\/p>\n\n\n\n

      \n
    1. Standard Library Functions:<\/strong> Predefined in C++<\/li>\n\n\n\n
    2. User-defined Function:<\/strong> Created by users<\/li>\n<\/ol>\n\n\n\n

      In this tutorial, we will focus mostly on user-defined functions.<\/p>\n\n\n\n

      \n\n\n\n

      C++ User-defined Function<\/h2>\n\n\n\n

      C++ allows the programmer to define their own function.<\/p>\n\n\n\n

      A user-defined function groups code to perform a specific task and that group of code is given a name (identifier).<\/p>\n\n\n\n

      When the function is invoked from any part of the program, it all executes the codes defined in the body of the function.<\/p>\n\n\n\n

      \n\n\n\n

      C++ Function Declaration<\/h3>\n\n\n\n

      The syntax to declare a function is:<\/p>\n\n\n\n

      returnType functionName (parameter1, parameter2,...) {\n    \/\/ function body   \n}<\/code><\/pre>\n\n\n\n
      Here’s an example of a function declaration.
       <\/p>\n\n\n\n
      \/\/ function declaration\nvoid greet() {\n    cout << \"Hello World\";\n}<\/code><\/pre>\n\n\n\n
      Here,<\/p>\n\n\n\n
        \n
      • the name of the function is greet()<\/code><\/li>\n\n\n\n
      • the return type of the function is void<\/code><\/li>\n\n\n\n
      • the empty parentheses mean it doesn’t have any parameters<\/li>\n\n\n\n
      • the function body is written inside {}<\/code><\/li>\n<\/ul>\n\n\n\n
        Note:<\/strong> We will learn about returnType<\/code> and parameters<\/code> later in this tutorial.<\/p>\n\n\n\n
        \n\n\n\n
        Calling a Function<\/h3>\n\n\n\n
        In the above program, we have declared a function named greet()<\/code>. To use the greet()<\/code> function, we need to call it.<\/p>\n\n\n\n
        Here’s how we can call the above greet()<\/code> function.<\/p>\n\n\n\n
        int main() {\n     \n    \/\/ calling a function   \n    greet(); \n\n}<\/code><\/pre>\n\n\n\n
